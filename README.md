# cinema-website-records
本仓库用于系统化记录互联网上出现的各类影视相关网站信息，重点整理电影与电视剧站点的基础资料结构。通过统一格式记录站点名称、访问地址及站点类别，方便进行后续的数据比对与信息查阅。所有收录内容均来自公开网络环境，不包含任何站内视频资源或播放接口。本项目的建立初衷是用于站点信息整理与行业观察，不参与任何形式的内容传播、推广或商业合作。仓库所列信息仅作参考用途，不构成使用建议或风险提示，站点访问行为需由用户自行判断。
